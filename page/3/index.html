dependencies和devDependencies

发表于 2016-11-01   |   分类于 学习笔记   |   阅读次数

在使用npm时，我们总能看到packjson里有dependencies和devDependencies

  • dependencies 使用npm i module –save 安装
  • devDependencies 使用 npm i module –save-dev 安装

packagejson官方文档

参考官方文档可以大概理解成

  • dependencies 为生产环境所需依赖，不包含测试或过渡性依赖
  • devDependencies 开发调试过程中集成测试等依赖
